# 🗒️ Notes-React

A sleek, responsive, and minimalist note-taking application built with **React 19**, **Vite**, and **Tailwind CSS**. Highlighting a modern dark UI with interactive sticky-note styled cards.

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://notes-react-coral-three.vercel.app/)
[![React](https://img.shields.io/badge/React-19.0.0-blue.svg)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-7.3.1-646CFF.svg)](https://vitejs.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.2.1-38B2AC.svg)](https://tailwindcss.com/)

---

## 🚀 Key Features

- **Intuitive UI:** Minimalist dark mode design for focused note-taking.
- **Dynamic Note Creation:** Instantly add notes with a title and detailed description.
- **Interactive Sticky Notes:** Each note is displayed on a classic sticky-note themed card.
- **Effortless Management:** Delete notes with a single click.
- **Fully Responsive:** Optimized for both desktop and mobile viewing.
- **Fast Performance:** Powered by Vite for near-instant load times and rapid development.

## 📦 Getting Started

Follow these steps to set up the project locally:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation
Clone the repository and install dependencies:

```bash
# Clone the repository
git clone https://github.com/sahilshingate01/Notes-react.git

# Navigate into the directory
cd Notes-react

# Install dependencies
npm install
```

### 3. Running the App
Start the development server:

```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### 4. Build for Production
To create an optimized production build:

```bash
npm run build
```
