# React Web App

A simple React web application built using React.js.
This project is created for learning and practice purposes.

## 🚀 Features

* Built with React
* Component-based structure
* Clean UI
* Fast development setup

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/sahilshingate01/Notes-react.git
```

Go to project folder

```bash
cd Notes-react
```

Install dependencies

```bash
npm install
```

## ▶️ Run Project

```bash
npm start
```

App will run on

```
http://localhost:3000
```

## 🏗️ Build for production

```bash
npm run build
```
